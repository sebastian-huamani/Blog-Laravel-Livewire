<p align="center"> Despues de clonar un repositorio de laravel necesitaremos hacer algunos pasos adicionales para que todo valla bien. </p>



## Recordatorio

```sh
- npm install
- composer install
- cp .env.example .env
- php artisan key:generate
- php artisan migrate --seed
```
